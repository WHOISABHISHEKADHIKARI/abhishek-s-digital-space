# Portfolio Section Images

Use these folders as your manual image workspace. Anything inside `public` is served from the site root, so:

- `public/sections/about/images/photo.jpg` becomes `/sections/about/images/photo.jpg`
- `public/sections/projects/images/himalaya-krishi.jpg` becomes `/sections/projects/images/himalaya-krishi.jpg`

Section folders:

- `about/images` - profile photo
- `experience/images` - company or organization logos
- `projects/images` - project screenshots and thumbnails
- `volunteering/images` - event photos, posters, certificates
- `blog/images` - blog cover images
- `news/images` - press, awards, recognition images
- `certifications/images` - certificate images
- `recommendations/images` - recommender avatars or screenshots

After adding an image, reference it from `abhishek_profile.json` using the public path.
