# Projects — Screenshots & Thumbnails

Drop project images here. Name each file after the project, e.g.:

  himalaya-krishi.jpg
  dev-community-nepal.jpg
  trailokya-builders.jpg

Then update the matching project entry in `abhishek_profile.json`:
  "imageUrl": "/sections/projects/himalaya-krishi.jpg"

Recommended: 1200×630px (16:9 ratio) for a clean card thumbnail.
If left empty, the page auto-fetches a live screenshot via Microlink.
