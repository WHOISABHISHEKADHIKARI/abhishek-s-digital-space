# News & Recognition — Press Clips & Badges

Drop press screenshots, award certificates, or recognition badges here. Example:

  images/top-10-contributor-prompt-eng.jpg
  images/hult-prize-panelist.jpg
  images/hric-steam-organizer.jpg

Then update the matching newsMedia entry in `abhishek_profile.json`:
  "imageUrl": "/sections/news/images/top-10-contributor-prompt-eng.jpg"

Recommended: any size — the cards will auto-scale the image.
