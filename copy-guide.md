# Copy Guide — Abhishek Adhikari Portfolio

## Voice Profile

| Dimension | Setting |
|-----------|---------|
| Formality | Casual |
| Humor | Funny (warm) |
| Respect | Irreverent (confident) |
| Enthusiasm | Enthusiastic |

## Tone by Context

| Context | Tone | Example |
|---------|------|---------|
| Hero / Intro | Confident, grounded | "Agritech entrepreneur and community builder" |
| Experience | Direct, matter-of-fact | "Selected roles" — not a boast, just the facts |
| Projects | Excited, curious | "Live work and ventures" |
| Certifications | Proud but humble | "Proof and recognitions" |
| Contact | Warm, open | "My inbox is open" |
| Error | Direct, helpful | "Something didn't load. Try again." |
| Empty | Encouraging | "Blank canvas. Time to create." |
| Success | Brief, confident | "Message sent." Not "Opened in mail app!" |

## CTA Patterns

| Context | Label | Notes |
|---------|-------|-------|
| Primary social | "LinkedIn" / "GitHub" | Platform name, not generic "Contact" |
| Form submit | "Send Message" | Specific, personal |
| External link | "Verify" | For certificate credentials |
| Certificate link | "View" | Opens in new tab |
| Show more | "Show N more" | Specific count builds trust |

## Microcopy Rules

- **No exclamation marks** in success messages. "Message sent" not "Message sent!"
- **No "Oops!"** errors. "Something didn't load. Try again."
- **No passive voice.** "We couldn't save your changes" not "Mistakes were made."
- **Sentence case everywhere.** "Send message" not "Send Message" (overrides current code — fix).
- **No AI cliches.** Never use "Elevate", "Seamless", "Next-Gen", "Game-changer".

## Form Copy

| Field | Label | Placeholder | Helper |
|-------|-------|-------------|--------|
| Name | Name | Your name | — |
| Email | Email | you@example.com | Used to reply to your message |
| Message | Message | What's on your mind? | — |

## Empty States

| Context | Copy |
|---------|------|
| No projects shown | "Nothing to show yet. Check back soon." |
| No certifications | "No certificates listed yet." |
| Image failed to load | Falls back to styled initials / placeholder |
| No search results | (not applicable — no search on this page) |

## Toast / Feedback

| Event | Message | Duration |
|-------|---------|----------|
| Form sent | "Message sent. I'll get back to you soon." | 4s |
| Form error | "Couldn't send. Check your connection and try again." | 6s |
| Copy to clipboard | "Copied" | 2s |
